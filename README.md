# Mini Boutique

Application React de catalogue de produits réalisée avec Vite.

## Prérequis

- [Node.js](https://nodejs.org/) v18+
- npm

## Installation

```bash
npm install
```

## Lancer le projet

```bash
npm run dev
```

Ouvrez ensuite [http://localhost:5173](http://localhost:5173) dans votre navigateur.

## Structure du projet

```
src/
├── components/
│   ├── Header.jsx         # En-tête de la boutique
│   ├── Footer.jsx         # Pied de page
│   ├── ProductList.jsx    # Liste de tous les produits
│   ├── ProductCard.jsx    # Carte d'un produit
│   ├── ProductBadge.jsx   # Badge de catégorie
│   └── ProductDetails.jsx # Produit mis en avant
├── App.jsx                # Composant racine
├── main.jsx               # Point d'entrée
└── index.css              # Styles globaux
```
